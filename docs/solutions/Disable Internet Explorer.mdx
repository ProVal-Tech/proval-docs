## Purpose

The purpose of this solution is to detect and disable Internet Explorer from Windows Computers.

## Associated Content

| Content                                                                                                                                                            | Type            | Function                                                                                                                                                                                                                       |
|--------------------------------------------------------------------------------------------------------------------------------------------------------------------|-----------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [CWM - Automate - Role - Internet Explorer Installed](https://proval.itglue.com/DOC-5078775-10731232)                                                            | Role            | The purpose of this Role is to detect the machines where Internet Explorer Feature is enabled.                                                                                                                               |
| [CWM - Automate - Internal Monitor - ProVal - Development - Internet Explorer Detected [Role]](https://proval.itglue.com/DOC-5078775-10731230)                  | Internal Monitor | The Purpose of this Internal Monitor is to Detect the online machines where [CWM - Automate - Role - Internet Explorer Installed](https://proval.itglue.com/DOC-5078775-10731232) role is detected.                      |
| [CWM - Automate - Script - Internet Explorer - Disable [Global, DV, Autofix]](https://proval.itglue.com/DOC-5078775-10220659)                                   | Script          | The purpose of this script is to disable the Internet Explorer.                                                                                                                                                             |
| ~Autofix - Disable Internet Explorer                                                                                                                              | Alert Template   | This alert template is responsible for executing [CWM - Automate - Script - Internet Explorer - Disable [Global, DV, Autofix]](https://proval.itglue.com/DOC-5078775-10220659) script all the machines detected by [CWM - Automate - Internal Monitor - ProVal - Development - Internet Explorer Detected [Role]](https://proval.itglue.com/DOC-5078775-10731230) monitor set. |

```

```
