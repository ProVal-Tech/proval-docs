## Purpose

The purpose of the solution is to monitor the hardware alerts generated on an HP server. The tools included are `HPEiLO` and `HPE Insight Management Agents`.

## Associated Content

| Content                                                                 | Type            | Function                                                                                                            |
|-------------------------------------------------------------------------|-----------------|---------------------------------------------------------------------------------------------------------------------|
| [Monitor - Missing Vendor Admin Software](https://proval.itglue.com/DOC-5078775-8052311) | Internal Monitor | Detects the HP servers which are missing the `HPE Insight Management Agents` application.                          |
| [Remote Monitor - HP Hardware Monitoring](https://proval.itglue.com/DOC-5078775-14961987) | Remote Monitor   | Detects the event logs generated by the `HPE Insight Management Agents` application for the HP Servers.            |
| △ Custom - Ticket Creation - Computer                                    | Alert Template   | Executes the [Script - Ticket Creation - Computer](https://proval.itglue.com/DOC-5078775-9098338) script against the machines detected by the above-mentioned monitor sets. |
| [Script - Deploy HP Insight Management Agent](https://proval.itglue.com/DOC-5078775-15049493) | Script           | Can be used to deploy the `HPE Insight Management Agents` application for the HP Servers.                          |
| [Solution - HP iLO Health Check](https://proval.itglue.com/DOC-5078775-10073297) | Solution         | Gather information about and monitor the health status of HP iLO devices.                                         |

## Implementation

1. Import the [Monitor - Missing Vendor Admin Software](https://proval.itglue.com/DOC-5078775-8052311) internal monitor from the `ProSync` plugin.

2. Import the [Script - Deploy HP Insight Management Agent](https://proval.itglue.com/DOC-5078775-15049493) script from the `ProSync` plugin.

3. Import the `△ Custom - Ticket Creation - Computer` alert template from the `ProSync` plugin.

4. Implement the [Remote Monitor - HP Hardware Monitoring](https://proval.itglue.com/DOC-5078775-14961987) remote monitor by following the implementation steps:
   - [Implement - Remote Monitor - HP Hardware Monitoring](https://proval.itglue.com/DOC-5078775-14979993)
     - Set the alert template of the monitor to `△ Custom - Ticket Creation - Computer`

5. Reload the System Cache.
   ![Reload the System Cache](../../static/img/HP-Servers-Monitoring/image_1.png)

6. Configure the solution as follows: 
   - Navigate to Automation → Monitors within the CWA Control Center and setup the following:
     - [Monitor - Missing Vendor Admin Software](https://proval.itglue.com/DOC-5078775-8052311)
       - **IMPORTANT:** View the results of the monitor before enabling. If there are more than 20 results, confirm with the consultant if they would like it enabled.
         - Configure with the alert template: `△ Custom - Ticket Creation - Computer`
         - Right-click and Run Now to start the monitor

### Optional Content:

1. Implement the [HP iLO Health Check](https://proval.itglue.com/DOC-5078775-10073297) solution by following the implementation steps provided in the [Solution](https://proval.itglue.com/DOC-5078775-10073297) document.
   - Implement this content **only if requested** by the consultant.

## Deprecation Process

1. Remove this content from the environment: 
   1. [Script - Deploy HP Insight Management Agent](https://proval.itglue.com/DOC-5078775-15049493)
      - You should be able to sync the deprecated content from the plugin if that is easier.
   2. Run the following SQL to remove the [Remote Monitor - HP Hardware Monitoring](https://proval.itglue.com/DOC-5078775-14961987) - This should be run through a RAWSQL monitor.
      ```sql
      Delete from groupagents where `Name` = 'ProVal - Production - HP Hardware Monitoring';
      Delete from agents where `Name` = 'ProVal - Production - HP Hardware Monitoring' and computerid > 0;
      ```
