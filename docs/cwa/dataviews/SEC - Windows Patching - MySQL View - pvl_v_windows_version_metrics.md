## Summary

The purpose of this view is to serve the [SEC - Windows Patching - Dataview - Windows 10 -11 Version Metrics](https://proval.itglue.com/DOC-5078775-11560727) (Clientid = 0) and [SEC - Windows Patching - Dataview - Windows 10 -11 Version Metrics - Client Filter](https://proval.itglue.com/DOC-5078775-11560609) (Clientid > 0) dataviews.

Unsupported for MySQL version < 5.7

## Dependencies

- [SEC - Windows Patching - Custom Table - plugin_proval_windows_os_support](https://proval.itglue.com/DOC-5078775-7780690)
- [CWM - Automate - Script - MySQL - View - Create (pvl_v_windows_version_metrics)*](https://proval.itglue.com/DOC-5078775-11564063)
