## Summary

This script installs the HP Insight Management Agent on Windows HP machines. The script downloads the installer from [Software Details - HPE Insight Management Agents for Microsoft Windows Server x64 Editions | HPE Support](https://support.hpe.com/connect/s/softwaredetails?language=en_US&softwareId=MTX_0084b7dc6d46464b8f7941db73).

## Global Parameters

| Name                     | Required | Example | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
|--------------------------|----------|---------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| TicketCreationCategory    | False    | 298     | This global variable stores the ID of the `Ticket Category` to create the ticket. The default value is `0`, which means the script will not create any tickets for failures. To enable the ticket creation feature of the script for failures, this global variable must be configured with the ID of a ticket category. Ticket Categories are used to control the CW Manage service board of the tickets generated by Automate. Navigate to the **System Dashboard --> Config --> Information Base Categories** to find the ID of the required ticket category. ![Ticket Category](../../../static/img/docs/cb0d8de7-88e2-4406-b2fb-7a7ee338ce37/image_2.webp) |
