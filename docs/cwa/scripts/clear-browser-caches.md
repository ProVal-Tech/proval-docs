## Summary

This script will clear the caches for the following browsers:

- Google Chrome
- Microsoft Edge
- Brave Browser
- Mozilla Firefox
- Microsoft Edge (Legacy)

**Note:** This is strictly for the browser cache and will not remove the browser history.
