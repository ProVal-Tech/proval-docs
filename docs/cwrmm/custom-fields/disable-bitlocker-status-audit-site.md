## Summary

Flag this custom field to exclude the site from the [BitLocker Status and Recovery Key Audit](/docs/b2a974b2-c231-4197-a639-d0775d77d7c7) solution. This custom field manages the auto-execution of the task. However, the [BitLocker Status and Recovery Key Audit](/docs/9682b5a8-d821-43f6-9b77-59d43b6ef015) task can still be run manually, independent of this field.
