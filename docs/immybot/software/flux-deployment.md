## Description

This script assists in performing the f.lux installation and uninstallation.
This is a client-specific product that downloads the `flux-setup.exe` using the URL `https://justgetflux.com/flux-setup.exe`.
The setup file or URL does not have the version details that can be used for the dynamic deployment.
The version required to be manually updated for a new upgrade for the software upgrade.
